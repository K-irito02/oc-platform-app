# � 现代化主题设计系统

—— 打造简洁、高效、美观的用户界面体验

基于现代设计原则和 Ant Design 5.x 组件库，构建一套符合当代审美和使用习惯的主题系统。注重用户体验、视觉层次和交互一致性，为 Qt 产品发布平台提供专业、可靠的前端界面。

---

## 一、设计理念

### 1.1 核心原则

- **简洁至上**：去除冗余元素，突出核心功能
- **一致性**：统一的视觉语言和交互模式
- **可访问性**：符合 WCAG 2.1 AA 标准，支持键盘导航和屏幕阅读器
- **响应式**：适配桌面、平板、移动等多种设备
- **性能优先**：流畅的动画和快速的加载体验

### 1.2 设计目标

| 目标 | 具体实现 |
|------|---------|
| **专业性** | 使用标准化的组件和布局，体现企业级产品品质 |
| **易用性** | 清晰的信息架构和直观的操作流程 |
| **美观性** | 现代化的色彩搭配和精致的视觉细节 |
| **可维护性** | 基于设计系统的组件化开发模式 |

## 四、响应式设计

### 4.1 断点定义

| 断点 | 宽度范围 | 设备类型 | 布局调整 |
|------|---------|---------|---------|
| `xs` | < 576px | 手机竖屏 | 单列布局 |
| `sm` | 576px – 768px | 手机横屏 | 单列，增大间距 |
| `md` | 768px – 992px | 平板 | 双列布局 |
| `lg` | 992px – 1200px | 小桌面 | 三列布局 |
| `xl` | 1200px – 1600px | 大桌面 | 四列布局 |
| `xxl` | ≥ 1600px | 超大屏 | 最大宽度限制 |

## 五、动画系统

### 5.1 常用动画

```css
/* ===== 淡入动画 ===== */
@keyframes fade-in {
  from { opacity: 0; }
  to   { opacity: 1; }
}

/* ===== 上滑入场 ===== */
@keyframes slide-up {
  from { 
    opacity: 0; 
    transform: translateY(20px); 
  }
  to   { 
    opacity: 1; 
    transform: translateY(0); 
  }
}

/* ===== 缩放入场 ===== */
@keyframes scale-in {
  from { 
    opacity: 0; 
    transform: scale(0.9); 
  }
  to   { 
    opacity: 1; 
    transform: scale(1); 
  }
}

/* 使用示例 */
.fade-enter    { animation: fade-in 0.3s ease-out; }
.slide-up-enter { animation: slide-up 0.4s ease-out; }
.scale-enter    { animation: scale-in 0.2s ease-out; }
```

## 六、组件设计规范

### 6.1 按钮设计

| 类型 | 用途 | 样式 |
|------|------|------|
| **主要按钮** | 主要操作，如提交、确认 | 蓝色背景，白色文字 |
| **次要按钮** | 次要操作，如取消、返回 | 白色背景，蓝色边框 |
| **危险按钮** | 删除、移除等危险操作 | 红色背景，白色文字 |
| **文本按钮** | 辅助操作，如查看详情 | 无背景，蓝色文字 |

### 6.2 表单设计

- **标签位置**：统一使用顶部标签布局
- **输入框高度**：40px（中等尺寸）
- **错误提示**：红色文字 + 图标
- **帮助文本**：灰色小字
- **必填标识**：红色星号

### 6.3 卡片设计

- **圆角**：12px
- **阴影**：轻微阴影，营造层次感
- **内边距**：24px
- **标题**：16px，中等字重
- **内容**：14px，常规字重

## 七、实施指南

### 7.1 开发规范

1. **使用设计令牌**：优先使用 CSS 变量，避免硬编码
2. **组件化开发**：基于 Ant Design 组件进行封装
3. **响应式优先**：移动端优先的设计思路
4. **性能考虑**：避免过度使用动画和特效

### 7.2 代码示例

```css
/* 使用设计令牌 */
.my-component {
  padding: var(--space-4);
  border-radius: var(--radius-base);
  background-color: var(--bg-white);
  box-shadow: var(--shadow-base);
  transition: all var(--duration-base) var(--ease-out);
}
```
